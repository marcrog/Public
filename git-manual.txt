"git clone https://github.com/...."
serve per importare un progetto github

una volta che importo un progetto, se ho autorizzazione per moficarlo posso poi 
salvare le modifiche facendo:

1) git add . (aggiunge tutte le modifiche fatte al "pacchetto")
2) git commit -m "descrizione pacchetto"
3) git push (spedisco pacchetto con la sua descrizione)

per aggiornare un progetto github salvato in locale
"git pull"



AUTENTICAZIONE:
github richiede un username e tocke ogni volta che si vuole fare un cambiamento 
ad una repositories

per non dover per ogni "push" fare questo procedimento si può salvare il token
sul dispositivo, in due modi:

1) git config --global credential.helper 'cache --timeout=28800'
si salva in RAM per un certo periodo di tempo (es. 8 ore)

2) git config --global credential.helper store
si salva permanentemente sul dispositivo
